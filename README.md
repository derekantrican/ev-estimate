# EV Range Tracker

A simple mobile-first React app for tracking the difference between your electric vehicle's estimated range and the actual distance to charging stations during road trips.

## Features

- Track Google Maps distance estimates vs. EV range estimates
- Organize data into "legs" for different trip segments
- Export data to CSV for analysis
- Dark theme optimized for mobile use
- Data persisted locally in browser

## Usage

```bash
npm install
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

To access from your phone on the same WiFi network, use your computer's IP address (e.g., http://192.168.0.25:3000).

## Build for Production

```bash
npm run build
```
